Application bureau en electron qui prend en argument une url. Qui la charge, garde en session
tout ce qui est cookie, localstorage etc… Et à sa réouverture charge la dernière page
visité sur ce nom de domaine. On peut aussi conserver la taille et positionnement
de la fenêtre, renommer la fenêtre, configurer un logo adapté… Cette appli permet
de faire un client lourd pour n’importe quel site .



Lancement de l'application avec un argument (url) :

npm start -- https://example.com
